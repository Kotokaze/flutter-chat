## Firebase Local Emulator Suite
```bash
$ docker-compose run
$ docker-compose exec firebase bash
root@---:/opt/workspace# firebase login --no-localhost
root@---:/opt/workspace# firebase init
root@---:/opt/workspace# firebase emulators:start
```
